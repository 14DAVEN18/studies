Operador BindCallback

Convierte la respuesta de una API basada en callbacks en un observable rxjs para manejar la respuesta de manera más flexible.

Dele una función f de tipo f(x, callback) y va a returnar una función g que cuando se llama como g(x) tendrá como salida un observable.

bindCalback no es un operador porque sus entradas y salidas no son observables. La entrada es una función func con algunos parametros. El último parametro debe ser una función callback que func llama cuando termina.

Ejemplo: Se usa bindCallback para crear un observable de la respuesta de un file reader que devuelve el contenido del archivo de texto plano o un error (en caso de que haya alguno). Se suscribe al observable resultante por medio del cual se asigna el contenido del texto a una variable que es mostrada en el documento html.
