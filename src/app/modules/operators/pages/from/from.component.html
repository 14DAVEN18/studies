Operador from

Crea un observable a partir de un arreglo, un objeto de tipo arreglo, una promesa, un objeto iterable, o un objeto de tipo observador.

from convierte varios otros objetos y tipos de datos en observables. También convierte una promesa, un tipo arreglo o un objeto iterable en un observable que emite los elementos en esa promesa, arreglo o iterable. Un string, en este contexto, es tratado como un arreglo de caracteres. Los objetos de tipo observable (contiene una función nombrada con el Símbolo ES2015 para observables) también pueden ser convertidos por medio de este operador.

Ejemplo: Se usa el operador from para crear una observable a partir de un arreglo de números
