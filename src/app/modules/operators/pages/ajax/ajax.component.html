Operador Ajax

Crea un observable para una solicitud Ajax con un objeto de solicitud con url, encabezados, etc o un string para una URL.

Ejemplo: Se usa el operador ajax para hacer una petición HTTP de tipo GET a una url, cuando el servidor responde, el método devuelve la respuesta envuelta en un observable y agrega un delay de 2000 ms, esto permite a la aplicación procesar la información de manera reactiva.

Nota 1: Una solicitud AJAX (Asynchronous Javascript and XML) es una forma para que una aplicación web se comunique con un servidor en segundo plano, sin recargar toda la página web. Esta técnica permite aplicaciones web dinámicas, rápidas y reponsivas mediante la actualización de partes de una página web asíncronamente. Nota 2: AJAX envuelve la respuesta en un objeto response
